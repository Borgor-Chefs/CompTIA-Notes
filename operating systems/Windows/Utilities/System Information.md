Program Name: `msinfo32.exe`

# System Information

This shows almost every single detail possible about your system. Including:

- hardware components
	- DMA
	- forced hardware
	- I/O
	- memory
- components
	- CD-ROM
	- display
	- infrared
	- input
	- modem
	- network
	- storage
	- ports
	- multimedia
- software environment
	- system drivers
	- print jobs
	- running tasks

The "System Summary" tab displays:

- OS name
- version
- OS manufacturer
- system name
- processor
- BIOS mode
- etc
